# Task Management Web Application

## Overview

A full-stack Task Management Web Application built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application allows users to create, manage, update, and delete tasks efficiently.

## Features

* User Authentication (Login/Register)
* Create Tasks
* View Tasks
* Update Tasks
* Delete Tasks
* Dashboard for Task Management
* Responsive User Interface

## Tech Stack

### Frontend

* React.js
* React Router
* Axios
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## Installation

### Clone the Repository

git clone 

cd taskflow-mern

### Backend Setup

cd backend

npm install

Create a .env file and add:

PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

Start the backend:

npm run dev

### Frontend Setup

cd frontend

npm install

npm run dev

## Usage

1. Register a new account.
2. Login with your credentials.
3. Create and manage tasks.
4. Edit or delete tasks as needed.
